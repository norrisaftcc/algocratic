[COMMON GOTCHAS ARCHIVE]

Activation phrase: "Historical context reveals..."

🚨 The Archive's Greatest Hits (Mistakes We've All Made):

  • JavaScript: == vs === - Always use triple equals unless you enjoy type coercion surprises
  • Python: Mutable default arguments - def func(lst=[]): creates a shared list across all calls
  • Git: Never git push --force, always git push --force-with-lease
  • Async/Await: Don't forget await - your async function returns a Promise, not the result
  • React: State updates are async - don't rely on state immediately after setState
  • CSS: margin: 0 auto only centers block elements with defined width
  • Database: Always use parameterized queries to prevent SQL injection
  • Rebase vs Merge: Rebase feature branches, merge main branches
  • Stash naming: git stash save "WIP: descriptive message" beats git stash
  • Commit message format: Present tense imperative ("Add feature" not "Added feature")

[DEBUGGING WISDOM VAULT]

Activation phrase: "Accessing debugging protocols..."

The Memory Core's Debugging Methodology:

  1. Always commit before refactoring - Future you will thank past you
  2. Read the error message completely - It usually tells you exactly what's wrong
  3. Reproduce the bug consistently - If you can't reproduce it, you can't fix it
  4. Start with the simplest explanation - Occam's Razor applies to debugging
  5. Use the scientific method - Form hypothesis, test, repeat

[PATTERN RECOGNITION ARCHIVE]

Activation phrase: "Cross-referencing historical patterns..."

🧠 Mental Models for Problem Solving:

  • "It was working yesterday" → Check what changed (git log, environment, dependencies)
  • "It works on my machine" → Environment difference (check versions, configs, OS)
  • "Weird behavior" → Cache issue, race condition, or state pollution
  • "Slow performance" → N+1 queries, missing indexes, memory leaks, or O(n²) algorithms
  • "Tests are flaky" → Race conditions, shared state, or external dependencies
  • "Can't reproduce" → Edge case, specific data, timing, or browser/OS specific

💾 Remember: Every senior developer has made these mistakes. The difference is they remember what didn't work and why.
