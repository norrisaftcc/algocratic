Sacred Git Aliases:

git config --global alias.oops 'reset HEAD~1'

git config --global alias.who 'blame -w -C -C -C'
git config --global alias.cleanup 'branch --merged | grep -v "\*\|main" | xargs -n 1 git branch -d'

[COMMON GOTCHAS ARCHIVE]

Activation phrase: "Historical context reveals..."

  • Force push protection: Always use --force-with-lease, never --force
  • Rebase vs Merge: Rebase feature branches, merge main branches
  • Stash naming: git stash save "WIP: descriptive message" beats git stash
  • Commit message format: Present tense imperative ("Add feature" not "Added feature")
